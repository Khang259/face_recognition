Chạy dự án local
- Kiểm tra sử dụng uv `uv --version`
-> Nếu chưa cài sử dụng `pip install uv`
- Khi pull code sử dụng `uv sync`
- Thêm lib `uv add <libs>`

- Chạy dự án không cần activate venv sử dụng `uv run python <file>`
